#ANGULAR COMMANDS

- Create new angular app
=>  ng new <app-name>

- Run angular app
=>  ng serve

- Run angular app and open in default browser
=>  ng serve - open
         OR
    ng serve - o 

- Create new angular component
=>  ng generate component <component-name>
                OR
    ng g c <component-name>

- Create an angular component without html file
=>  ng g c <component-name> --inline-template

- Create an angular component without styles file
=>  ng g c <component-name> --inline-style

- Create an angular component without html and styles file
=>  ng g c <component-name> --inline-template --inline-style

- Create an angular component without .spec.ts file
=>  ng g c <component-name> --skip-tests
